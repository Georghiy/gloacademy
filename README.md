из 1 
FormData = https://learn.javascript.ru/formdata

let formData = new FormData([form]) - используются, чтобы взять данные из HTML-формы и отправить их с помощью fetch или другого метода для работы с сетью. Автоматически формирует объект и помещает в него значения из инпута, основываясь на атрибуте name {input.name : input.value}
Его особенность заключается в том, что методы для работы с сетью, например fetch, позволяют указать объект FormData в свойстве тела запроса body. 

let response = await fetch('/article/formdata/post/user-avatar', {
      method: 'POST',
      body: new FormData(formElem)
    });

Он будет соответствующим образом закодирован и отправлен с заголовком Content-Type: multipart/form-data.
То есть, для сервера это выглядит как обычная отправка формы.

Методы (set удаляет данные из объекта formData имеющие такое же название, append нет)
formData.append(name, value)
formData.append(name, blob, fileName) 
formData.set(name, value)
formData.set(name, blob, fileName)

указание аргумента blob, позволяет передать в объете файлы как blob данные (т.е. к примеру изображение или нали)

new Set() — это конструктор для создания коллекции, хранящей только уникальные значения любого типа (примитивы или объекты). В отличие от массивов, Set автоматически удаляет дубликаты, а элементы упорядочены по времени добавления.
    const ganres = new Set()
    data.anime.forEach(item => {
      ganres.add(item.ganre)
    })
код перебирает всу элементы в data и записывает только уникальные значения в item.ganre
!!! Методы объекта new Set() уникальны и отличаются от методов массива или объекта
